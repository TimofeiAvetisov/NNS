# Структура
Пока что это просто для себя чтобы не забыть
### RandomGenerator
Просто класс для того чтобы генерировать матрицы с разными распределениями, Eigen может либо нормально на [-1, 1] либо с кастомным функтором. 
По-сути единственный полезный метод доступный извне init_linear_weights - даем размеры, какую схему использовать (Normaд, XavierNormal, HeNormal) и gain - нужен только для normal, в остальных не используется.
Так же можно вывести seed для отладки через show_seed
### TypeErasure 
Стиратель типов, в целом в файле написнно как его использовать
### Layers
Есть концептуальное отличие между LinearLayer и остальными, потому что первый всегда одинаковый, а остальные нужно создавать через TypeErasure.
